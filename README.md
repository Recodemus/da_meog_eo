# 🍽️ 다 먹어 (Da meog-eo) 
> **"오늘 뭐 먹지?" 고민 해결! 냉장고 재료 기반 맞춤형 레시피 추천 서비스**

![메인화면_스크린샷](https://via.placeholder.com/800x400?text=Da+Meog-eo+Main+Screen) ---

## 📌 프로젝트 개요
- **배경**: 1인 가구 증가에 따른 식재료 관리 미흡 및 음식물 쓰레기 발생 문제 해결
- **목표**: 냉장고 속 식재료 유통기한을 관리하고, 보유한 재료로 바로 요리할 수 있는 레시피를 추천하여 식재료 낭비 방지
- **핵심 가치**: 관리(유통기한) 🔄 추천(레시피) 🔄 기록(소비)의 선순환 구조 제공

---

## ✨ 핵심 기능
- **🧊 나의 냉장고 관리**: 식재료 등록 시 유통기한에 따른 D-Day 자동 계산 및 시각적 경고(빨강/노랑/초록) 제공
- **🧑‍🍳 맞춤 레시피 추천**: 내 냉장고 재료와 레시피 재료를 대조하여 **일치율이 높은 순**으로 정렬 및 **부족한 재료** 표시
- **⭐ 레시피 스크랩**: 마음에 드는 레시피를 보관하고 마이페이지에서 모아보기 가능
- **📱 모바일 친화적 UI**: 하단 탭 바를 이용한 앱 방식의 인터페이스와 비동기 통신(AJAX)을 통한 부드러운 사용감

---

## 🛠 기술 스택
- **Backend**: PHP 8.2 (PDO 사용)
- **Database**: MySQL (MariaDB)
- **Frontend**: HTML5, CSS3, JavaScript (Fetch API)
- **Server**: Apache (XAMPP Environment)
